# App Store Connect 隐私问卷参考

## 数据收集 / Data Collection

### 是否收集用户数据？
**否** - 我们不收集任何用户数据到我们的服务器

### 第三方数据共享（AI 功能）

**重要**：虽然我们不收集数据，但用户可以选择使用第三方 AI 服务。

#### 数据类型：其他使用数据 / Other Usage Data
- **收集目的**：应用功能（生成提交信息建议）
- **是否关联到用户身份**：否
- **是否用于追踪**：否
- **数据内容**：代码变更内容（git diff）
- **第三方共享**：是（用户选择的 AI 服务：OpenAI 或 Anthropic）
- **用户控制**：完全可选，需明确同意，可随时禁用

#### 数据类型：敏感信息 / Sensitive Info
- **收集目的**：应用功能（身份验证）
- **是否关联到用户身份**：否
- **是否用于追踪**：否
- **数据内容**：GitHub Personal Access Token、AI API Key
- **存储位置**：仅存储在用户设备的 macOS Keychain
- **第三方共享**：否（仅用于直接 API 调用）

## 隐私实践详情

### 网络访问说明
1. **Git 操作**：连接用户配置的 Git 服务器
2. **GitHub API**：使用用户的 Token 访问 GitHub
3. **AI 服务**：仅在用户启用并同意后连接

### 数据保护措施
- App Sandbox 隔离
- Keychain 加密存储
- HTTPS 加密传输
- 无服务器端数据收集

### 用户控制
- AI 功能完全可选
- 可随时删除凭证
- 可随时撤销权限
